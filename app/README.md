# Application Source

В этом каталоге находятся исходники приложения Podinfo, импортированные из upstream-репозитория:

https://github.com/stefanprodan/podinfo

В рамках дипломного проекта:
- исходный код приложения хранится в `app/`
- контейнеризация будет описана отдельно в `build/`
- Kubernetes-манифесты и Helm chart будут описаны отдельно в `deploy/`
- CI/CD пайплайны будут описаны отдельно в `.github/workflows/`
- инфраструктура будет описана отдельно в `infra/terraform/`

Таким образом, каталог `app/` содержит только приложение как исходный код.
