# Краткое описание дипломного проекта

## Что сделано
- выбран open-source проект Podinfo
- импортированы исходники приложения
- написан собственный Dockerfile
- настроен CI pipeline в GitHub Actions
- образы публикуются в GHCR
- написан Helm chart
- Terraform разворачивает инфраструктуру в Yandex Cloud
- поднят Kubernetes-кластер на k3s
- настроен CD pipeline
- подключён мониторинг Prometheus + Grafana
- подключено логирование Loki + Alloy
- приложение доступно через Интернет



## Основная идея
На базе Podinfo реализован полный DevOps-цикл: от исходного кода до автоматизированного деплоя в облачную инфраструктуру с мониторингом и логированием.
